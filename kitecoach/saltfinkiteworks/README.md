# KiteCoach Partner Kit Template

Editable printable HTML templates for kite school partner materials.

## Files

- `index.html` — 8.5 x 11 classroom poster
- `counter-card.html` — 8.5 x 11 foldable counter card
- `qr-assets.html` — QR usage sheet
- `landing-page.html` — partner landing page starter
- `styles.css` — brand colors and print styling
- `assets/` — placeholder logos and QR codes

## School customization

Ask each school for:

1. School logo, preferably SVG or PNG with transparent background
2. Primary brand color
3. Secondary brand color
4. Tertiary/accent brand color
5. School name exactly as they want it displayed
6. Preferred partner URL slug

Then update:

- `--primary`
- `--secondary`
- `--tertiary`
- school name text
- QR code URL
- school logo file

## Print

Open the HTML file in a browser and print to PDF or paper.

Recommended:
- Paper size: Letter / 8.5 x 11
- Margins: None or minimum
- Background graphics: enabled
