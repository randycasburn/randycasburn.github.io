Original Works

This product is derived from an open source project: PowerTrader_AI


I found it on GitHub and immediately became fascinated by its potential. The original is written in Python. I wanted to make it more accessible to non-technical users, so I created this Electron-based desktop application. The core AI algorithms and trading logic are mostly unchanged from the original project, which is why I can't claim credit for the "intelligence" behind it. My contribution is in rewriting it and packaging it into a user-friendly interface and adding features to make it easier to use.
